# Network Packet Sniffer (Python)

This project is a **Python-based Network Packet Sniffer** developed to understand the fundamentals of **computer networking and cybersecurity**. The tool captures and analyzes network packets in real time, allowing users to observe how data travels across a network.

The packet sniffer monitors network traffic and extracts important packet information such as **source IP address, destination IP address, protocol type, and packet size**. It helps demonstrate how communication occurs between devices using common protocols like **TCP, UDP, and ICMP**.

The goal of this project is to gain hands-on experience with **packet-level communication, TCP/IP networking concepts, and network traffic analysis**. It also serves as a learning project to explore how network monitoring tools work internally.

This project is built using **Python and networking libraries** and is intended for educational purposes to strengthen understanding of **network security, packet inspection, and protocol analysis**.

## Features

* Capture network packets in real time
* Extract source and destination IP addresses
* Identify network protocols (TCP, UDP, ICMP)
* Display packet size and basic packet details
* Support basic packet filtering for specific protocols

## Learning Objectives

* Understand how network packets travel across a network
* Learn the structure of IP packets and protocols
* Explore how packet sniffers capture and analyze traffic
* Gain practical experience with networking concepts using Python

## Disclaimer

This project is created **for educational and learning purposes only**. It should only be used on networks where you have permission to monitor traffic.
